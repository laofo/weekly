## 技术思路

**What's new in ES2022?**  
<https://dev.to/jasmin/whats-new-in-es2022-1de6>  
这个 await import 方法很是喜欢。

```js
let lodash;
try {
  lodash = await import('https://primary.example.com/lodash');
} catch {
  lodash = await import('https://secondary.example.com/lodash');
}
```
