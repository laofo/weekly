**带带弟弟：OCR 通用验证码识别**  
<https://github.com/sml2h3/ddddocr>  
中国人不骗中国人，API 的确很易用。

```python
import ddddocr

ocr = ddddocr.DdddOcr()

with open("test.jpg", 'rb') as f:
    image = f.read()

res = ocr.classification(image)
print(res)
```
